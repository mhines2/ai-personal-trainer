# ai-personal-trainer
## Project Goals
- User Profile Creation: Allow users to sign up and create a profile.
- Workout Generation: Provide daily workout plans based on user preferences.
- Fitness Advice: Offer tailored fitness tips and advice.
- Email Notifications: Send daily emails with workout plans.
## 1.2 Tech Stack
- Backend Framework: Flask (Python)
- AI Model: OpenAI API for generating workouts
- Database: SQLite (for development) or PostgreSQL (for production)
- Email Service: Flask-Mail or an external service like SendGrid
- Frontend: HTML/CSS/JavaScript for the web interface
